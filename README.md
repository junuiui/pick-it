# Key Features
## Real-time Group Sessions: Synchronized game states across all participants using WebSockets (Supabase Realtime).
## Live Presence: Real-time tracking of active users within a session.

### Interactive Mini-games:
- Ladder Game: Classic synchronized ladder climbing logic.
- Roulette: Customizable spinning wheel with shared results.
- Anonymous Vote: Private voting system with live result aggregation.
- Bomb Game: High-stakes synchronized timer game.

### Responsive UI/UX: Optimized for both mobile and desktop group settings.

### Tech Stacks
- UI & Icons
